# Bikestore-Analysis

 * Project Title *
   
Bike Store Sales Dashboard



 * Description 

Bike Store Sales Dashboard is visually engaging analytical power BI report to help users explore Bike store sales analytics. The dashboard highlights impact of various factors on revenue .The project is aimed at maximizing revenue and staying up to date with the store inventory so that there is no shortage of inventory when order is received. 



 * tech stack 

Power BI desktop : Main data visualization platform used for report creation.

power query : Data transformation and cleaning layer used for reshaping and preparing data.

DAX ( data analysis expression ) : used for calculated measures, dynamic visuals and conditional logic.

Data Modelling : relationship establishment among tables.

Drill through functionality : Used for dynamic visual creation such as store info.
